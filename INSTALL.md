Tutoriel d'installation de Kernel-Tools
1. Télécharger le ZIP
Télécharge le fichier Kernel-ToolsV3.zip.
2. Extraction
Fais un clic droit sur le fichier ZIP et clique sur "Extraire tout". Une fois extrait, ouvre le dossier Kernel-ToolsV3.
3. Lancement
Double-clique sur start.bat pour lancer l'installation et démarrer Kernel-Tools. Le script installera automatiquement Python et tous les modules nécessaires.
💡 Astuce : Si ton antivirus bloque le script, autorise-le manuellement car il peut s'agir d'un faux positif.

        English tutorial 

Kernel-Tools Installation Tutorial
1. Download the ZIP
Download the Kernel-ToolsV3.zip file.
2. Extraction
Right-click on the ZIP file and select "Extract All". Once extracted, open the Kernel-ToolsV3 folder.
3. Launch
Double-click on start.bat to start the installation and launch Kernel-Tools. The script will automatically install Python and all required modules.
💡 Tip : If your antivirus blocks the script, allow it manually as it may be a false positive.
